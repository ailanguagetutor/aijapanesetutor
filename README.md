## Getting Started

### Prerequisites

- Node.js
- Yarn package manager

### Installation

1. Clone the repository

2. Install dependencies:
```bash
yarn install
```

3. Start the development server:
```bash
export GEMINI_API_KEY=your_api_key_here
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
